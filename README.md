## Cómo ejecutar el proyecto
1. Descomprimir el archivo .zip.
2. Abrir el entorno de desarrollo NetBeans.
3. Ir a File > Import Project > From ZIP... (o abrir directamente la carpeta del proyecto descomprimida).
4. Localizar el archivo `Main.java` dentro del paquete `integrado.prog2`.
5. Hacer clic derecho sobre el archivo y seleccionar "Run File" (o presionar Shift + F6) para iniciar el menú en la consola.
